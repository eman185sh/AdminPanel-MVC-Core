# Admin Dashboard Task - ASP.NET Core MVC

This repository contains a dedicated Admin Panel implementation with a custom layout and sidebar navigation.

## 🛠️ Task Achievements
*   **Custom Admin Layout:** Created `_AdminLayout.cshtml` featuring a fixed sidebar and responsive content area.
*   **Admin Controller:** Implemented `AdminController.cs` to handle administrative routing.
*   **Dashboard View:** Developed a `Dashboard` action that displays system statistics.
*   **Dynamic UI:** Integrated a Bootstrap-styled data table for user management.

## 📂 Key Files
*   `Controllers/AdminController.cs` - Manages dashboard logic.
*   `Views/Shared/_AdminLayout.cshtml` - The master template for the admin area.
*   `Views/Admin/Dashboard.cshtml` - The main dashboard interface with the statistics table.

## 🚀 How to Access
1. Run the project in Visual Studio.
2. Navigate to: `https://localhost:XXXX/Admin/Dashboard`.
